Requerimientos Básicos
*****************************************************************
1. Servidor de Aplicaciones Web compatible con PHP 5
2. Servidor de Base de Datos MySQL 5
3. Navegador Web compatible con HTML5 Y css3



Instrucciones de instalación de 2 minutos.
*****************************************************************

1. Importar el archivo "DB_tutoriasJV.sql" a un gestor de Base de Datos MySQL
2. Editar el archivo "funciones.php" con los valores establecidos en su servidor de base de datos, así como el nombre de la base de datos importada en el paso anterior. Todo ello situado con la carpeta php.
3. Copiar y pegar la carpeta TutorApp en el directorio público correspondiente a su servidor de Aplicaciones Web.
4. Visitar la URL correspondiente a su Intranet, apuntando directamente al directorio raíz de TutorApp.
3. Listo...
